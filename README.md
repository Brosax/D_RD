# DEKRA R&D CLI

Welcome to the **DEKRA R&D** interactive CLI tool.

This repository contains the customized terminal coding assistant for DEKRA R&D, built on top of the reverse-engineered Claude Code CLI base.

## Features

- **Interactive AI Assistant**: A conversational REPL directly in your terminal.
- **Customized for DEKRA R&D**: Tailored startup screens and custom functionality.
- **Cross-Platform**: Works on macOS, Windows, and Linux.

## Installation

Ensure you have [Bun](https://bun.sh/) installed, then run:

```bash
bun install
```

## Running the Tool

To start the DEKRA R&D CLI in development mode:

```bash
bun run dev
```

To build the project for distribution:

```bash
bun run build
```
